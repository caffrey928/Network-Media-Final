# IOTA web crawler

## Installation

1. environment

```
pip install -r requirement.txt
```

2. Webdriver

- install webdriver [https://chromedriver.chromium.org/downloads](https://chromedriver.chromium.org/downloads)
- select right version of your chrome(more-help-more about chrome)

2. run

- modify desired addr

```
./main.py
```
